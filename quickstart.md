# Quick Start Guide

Get the AI Content Generator running in 5 minutes!

## 1. Install Python Dependencies

```bash
pip install -r requirements.txt
```

## 2. Get Your API Keys

You need three API keys:

| Service | Sign Up Link | Free Tier |
|---------|--------------|-----------|
| OpenAI | https://platform.openai.com/api-keys | $5 free credit |
| SerpAPI | https://serpapi.com/ | 100 searches/month |
| TextRazor | https://www.textrazor.com/ | 500 requests/day |

## 3. Run the App

```bash
streamlit run app.py
```

## 4. Enter Your Keys

When the app opens in your browser:
1. Look at the **left sidebar**
2. Enter all three API keys
3. Leave other settings as default
